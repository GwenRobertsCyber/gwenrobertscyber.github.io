Lab 03: Code a Secure Web Portfolio from Scratch

Time Estimate: 45-60 minutes

Objective

Learn to read and write fundamental HTML and CSS by building a static web page without relying on vulnerable third-party plugins or site builders.

Part 1: The Setup

We are avoiding heavy software for this lab. You only need a text editor and a web browser.

  • Create a new folder on your desktop called MyPortfolio.
  • Open your text editor (like Visual Studio Code or Notepad) and create a new file named index.html inside that folder.

Part 2: The HTML Skeleton

HTML (HyperText Markup Language) is the bone structure of the internet. We use "tags" to tell the browser what kind of content we are displaying. Type (do not copy/paste!) the following code into your index file to practice reading the syntax:

<html>
<body>
  <h1>My Name</h1>
  <p>I am learning cybersecurity and web development.</p>
</body>
</html>

Part 3: Adding Style with CSS

CSS (Cascading Style Sheets) is the paint and interior design. Without it, the web is just black text on a white background. Let's add a "style" section right above our body tag to make it look professional.

<style>
  body {
    background-color: #1a1a1d;
    color: #50c878;
    font-family: sans-serif;
  }
</style>

Part 4: Test Your Code

Save your index.html file. Now, simply double-click that file in your folder. It will open in your web browser. If you typed everything correctly, you should see a dark background with green text.

Homework Challenge

Now that you know how tags work, look up how to use an <a> tag and try adding a link to your favorite tech news site into your code!
